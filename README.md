# Jon Choi

Hi! This is my personal portfolio site showcasing my work as a Product Engineer. The site features a custom UI component library designed in Figma and implemented with React and Storybook, demonstrating my approach to building scalable, maintainable frontend systems. I focus on bridging design and engineering to create user-centered products that solve real problems.

### Getting Started ###
To view the components in Storybook, please follow these steps in your terminal:

1. Clone this repository: `git clone https://github.com/jisung594/personal-site`
2. Go to project root: `cd personal-site`
3. Install dependencies: `npm install`
4. Run the dev server: `npm start`
5. Run Storybook: `npm run storybook`
